# Simple Nest Backend API using GraphQl and MongoDB

- Schema first approach with nest and graphql
- npm run gt (Generate the types for schema files)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
